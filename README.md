# wide-receiver-predictor
Uses statistics and data analytics to forecast wide receiver fantasy performance — or at least see how close we can get in a sport ruled by chaos and small sample sizes
Predicting fantasy football performance is notoriously difficult due to small weekly sample sizes, injury volatility, and match-ups. This project aims not to guarantee perfect predictions, but to explore what insights can be gained from data trends and statistical modeling focused especially for wide receivers, where target share, game script, and variance can make or break weekly output.
